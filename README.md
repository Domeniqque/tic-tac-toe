* VueJs
* Async and await (promises)
* PWA
* WebWorkers

## To run

``` bash
After clone this project

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# run e2e tests
npm run e2e

# run all tests
npm test
```
